Nmap muss auf dem lokalen ebenfalls installiert werden. -> https://nmap.org/

Der Pfad des Nmap-Ordners muss kopiert werden. Bspw: C:\Program Files (x86)\Nmap\ 

Den Pfad bei [Mein PC]->[System Informationen]->[Erweiterte Einstellungen]->[Umgebungsvariablen] einfügen
PS: Bei einem selber und beim System hinzufügen!!! 

-> dann sollte Nmap in Python funktionieren


Falls nicht dann:
- "python-nmap" manuell deinstallieren
- Den Pfad bei [...]->[Umgebungsvariablen] einfügen
- Entwicklungsumgebung neu starten
- "python-nmap" neu installieren

-> dann sollte es hoffentlich funktionieren
